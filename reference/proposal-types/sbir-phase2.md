# SBIR Phase II

## Evaluator mindset
"Did Phase I succeed? Is there a real Phase III customer? Can this team deliver a prototype that transitions?"

## Required artifacts
- Phase I results summary (what was proven)
- Phase II technical plan (prototype development, testing, delivery)
- Commercialization plan with named Phase III customers / end users
- Transition letters of intent (if available — scoring lift)
- Past performance on prior SBIRs + commercialization record
- Phase II budget (larger, more detail than Phase I)
- Data rights: SBIR Policy Directive markings
